#### Запуск и Анализ
* Скомпилируйте и запустите:
```
go run main.go
```
После запуска в текущей директории появятся два файла: cpu.prof и mem.prof.

Анализируйте CPU-профиль:
```
go tool pprof cpu.prof
```

* Внутри интерактивной оболочки pprof вы можете ввести команду top (показать самые ресурсоемкие функции) или web (сгенерировать визуализацию в браузере, если установлен Graphviz).

* Анализируйте Профиль Памяти:
```
go tool pprof mem.prof
```
Здесь вы также можете использовать команду top или web, чтобы увидеть, какие функции вызвали наибольшее выделение памяти.
